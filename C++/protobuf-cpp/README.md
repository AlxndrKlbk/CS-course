CMake Protobuf Example
======================

Minimal example of using CMake and Conan to manage C++ project with protobuf, 

Build
======================
    
    $ mkdir build
    $ cd build
    $ python -m venv venv
    $ source venv/bin/activate
    $ pip install -r requirements.txt
    $ cmake ..
    $ make

Run
======================
    $ ./rpc
